# Modern Food Billing System

A simple yet powerful restaurant billing application built with Python. Designed for restaurants, cafés, and small food businesses, this system makes billing faster, easier, and more accurate with a modern, user-friendly interface.

## Features

* Modern and responsive interface powered by `ttkbootstrap` with Dark and Light mode support.
* Quick calculation of food and beverage orders.
* Automatic tax and service charge calculations.
* Built-in calculator for discounts, penalties, and quick calculations.
* Generates clear and professional receipts.
* Print receipts directly from the application.
* One-click reset option to start a new order instantly.

## Built With

* Python 3
* Tkinter
* ttkbootstrap

## How It Works

1. Enter the quantity of food and drink items.
2. Click **Total** to calculate the bill.
3. Apply discounts or additional charges if needed.
4. Generate a receipt with all order details.
5. Print or save the receipt for your records.
